Prohibited patterns

No throat-clearing — "this is important because"
No negative preamble — state what something is
No Not X, but Y — fold to the affirmative
No em-dashes in body text — restructure the sentence
No lists — integrate into prose
No passive voice unless actor is genuinely unknown
No academic register — write for informed general reader

How to Use This Hub

▶ End of Session

  1. Click Export at the top of the Notes tab — copies your data to clipboard.
  2. Open Claude in the Over-classified project → paste → say "Update the project hub."
  3. Download the new index.html Claude produces → save to iCloud Drive/Downloads.
  4. Press ⇧⌘U to run OC Hub Upload → uploads to GitHub → live in ~60 seconds.
  5. Refresh the hub on any device.

▶ Starting a Session

Open https://brownd11-a11y.github.io/oc-hub/ in Safari on any device. Bookmark on Mac; home screen icon on iPhone/iPad. Data loads automatically.

▶ Multi-Device

One device at a time — finish and sync before switching. If you made changes on multiple devices before syncing, paste all exports to Claude in one session and say "Merge these exports."

▶ Mac Shortcuts

  • ⇧⌘U — OC Hub Upload: uploads index.html from iCloud Drive/Downloads to GitHub.

To reassign: System Settings → Keyboard → Keyboard Shortcuts → Services → OC Hub Upload.

▶ Roadmap Gantt

  • Hidden — chart off
  • Bars + Gates — phase bars with gate diamonds at target dates
  • Bars + Gates + Notes — adds date/label annotation below each bar

Bar spans earliest start → latest end date across steps. Gate diamond sits at latest end date. Click a diamond to mark it met — records today automatically. Each step has Start and End date fields in the task list.

▶ Data Safety

  • Old localStorage keys deleted automatically — no accumulation.
  • Forgot to export? Work is still in localStorage on that device — open, export, give to Claude.
  • GitHub is source of truth. localStorage is a session buffer.
